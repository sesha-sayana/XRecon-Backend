# Xrecon

Xrecon based on XRPL

## Project Setup

1. Clone:

```
git clone https://github.com/sesha-sayana/XRecon-Backend.git
cd XRecon-Backend
```

2. Run cp .env.example .env file to copy example file to .env
   Then edit your .env file with DB credentials and other settings.

3. `npm install`

4. `npm start`
